# CNT-CVF Simulation - Claude Code 開発ガイド (Phase 2)

## プロジェクト概要

真空ろ過法（CVF: Centrifugal Vacuum Filtration）によるCNT配向プロセスのシミュレーション。実験条件と配向度（ラマンGピーク比）の関係を再現し、配向度を最大化する設計指針を出すことが目的。

最終的に出したい答え：
1. **長さ分布の最適形**（中央値・幅・カットオフ）
2. **CNT 投入量・濃度の最適値**（面密度を決める）
3. **流速（drop 間隔）の最適値とその物理的理由**
4. 未解明事項（短尺スタートで配向度 3.4、真空 2.5 で 1.1）の説明

---

## 重大な方針転換（Phase 1 → Phase 2）

### Phase 1（旧モデル、archive 化済み）
**前提**: 流体配向（Jeffery + Brownian）+ 多層 + 長さ分布の MC
**結論**: 達成可能範囲で予測上限 G ≈ 1.4。実測 1.78、3.4 を**原理的に説明できない**ことが判明。

### Phase 2（新モデル、本ガイドの対象）
**前提**: CVF の配向機構は **2D nematic 液晶相転移** であり、流体配向ではない。
**根拠**: 文献（Komatsu et al. 2020、He et al. 2016、Gao & Kono 2019、Speranza & Sollich 2002）で機構が確立されている。3D Onsager 相転移は CNT 濃度から否定される（要 ~5 mg/mL、実条件 ~15 µg/mL）。実際の機構は：
1. CNT が膜表面付近に蓄積し面密度 ρ_s が上がる
2. 2D Onsager 型の相転移が起きる（ρ_s × L² が普遍量）
3. 膜の溝が director の方向を決定（Komatsu 2020）
4. 蓄積が層状に進み、最終的に多層膜になる

参考文献:
- He, Gao, et al., *Nat. Nanotechnol.* 11, 633 (2016) — 元論文
- Komatsu, Kono, et al., *Nano Lett.* 20, 2332 (2020) — 溝アシスト配向
- Gao & Kono, *R. Soc. Open Sci.* 6, 181605 (2019) — レビュー
- Speranza & Sollich, *Phys. Rev. E* 67, 061702 (2003) — 多分散 Onsager
- Kawai, et al., *Appl. Phys. Express* 6, 065103 (2013) — ラマン G ピーク比

## 物理モデル（Phase 2）

### Layer 1: 蓄積過程
- CVF プロセスで CNT が膜表面に集まる
- 単位面積あたりの CNT 本数 ρ_s [本/m²] が時間とともに上がる
- 流体力学はここに登場（CNT を膜面に運ぶ駆動力として）
- **配向の駆動力ではない**

### Layer 2: 2D Onsager 相転移
- 蓄積した CNT が 2D 棒として相互作用
- 制御パラメータ: 無次元密度 b ∝ ρ_s × L × d
- 等方相 → nematic 相の転移
- 多分散の場合: 共存領域が広がる、長尺優先で nematic 化、短尺は等方相に取り残される
- 出力: 秩序パラメータ S = ⟨2cos²θ - 1⟩ と角度分布 P(θ)

### Layer 3: 溝による方向決定
- 膜の溝（W=334 nm 中心 / 300 nm 外周）が nematic director の方向を固定
- S が高い相が形成されれば大域配向、低ければ多 domain
- Step 4 以降で扱う

### Layer 4: 多層構造の積分
- 底層が最初に相転移、上層が後から堆積
- 各層の S を積算してラマン G 比を出す
- Step 4 以降で扱う

### G ピーク比の計算（Phase 1 から継承）
角度分布 P(θ) から:
```
I_∥/I_⊥ = ∫ P(θ) (10cos²θ + sin²θ) dθ / ∫ P(θ) (cos²θ + 10sin²θ) dθ
```
係数 10 はラマン偏光依存項（Kawai 2013）。`orientation.py` に実装済み、再利用する。

## 主要な物理量と式（Phase 2）

### 2D Onsager 理論

剛体棒（長さ L、直径 d）が 2D 平面に拘束された系。

**無次元密度**:
排除面積（2 本の棒が交差したときに排除される面積）は |sin(θ-θ')| × 2 L d（先端効果無視、L >> d）。これから 2D Onsager の自然な無次元化:
```
b = ρ_s × L × d
```
ρ_s は面密度 [本/m²]。

**自由エネルギー（単位本数あたり、単位 k_B T）**:
```
F/N = ⟨ln P(θ)⟩ + b × (2/π) × ⟨⟨|sin(θ-θ')|⟩⟩
```
- 第1項: 配向エントロピー（等方分布で最小）
- 第2項: 排除面積からくる相互作用（揃うほど小さい）
- ⟨⟨...⟩⟩ は P(θ) と P(θ') の二重平均

**相転移の臨界密度**:
2D Onsager の場合、b_c ≈ 3π/8 ≈ 1.18（Maier-Saupe 近似ベース、教科書値）。
正確には自由エネルギーから自己無撞着に解く。実装では:
- 角度分布 P(θ) を変分的に最適化
- Maier-Saupe 風の試行関数 P(θ) ∝ exp(α cos 2θ) で α を変分パラメータとする
- α=0 (S=0) と α>0 (S>0) を比較

無次元化を解くと b_c = const なので:
```
ρ_c ∝ 1/(L × d)
```
つまり **短尺・太バンドルは高密度が必要**。

### 多分散版（Step 2 以降）

長さ分布 P(L) があるとき、各長さの CNT が同じ角度分布 P(θ) を共有する近似（mean-field）:
```
F/N = ⟨ln P(θ)⟩ + ρ_s × ⟨L⟩ × d × (2/π) × ⟨⟨|sin(θ-θ')|⟩⟩
```
より厳密には、長さによって違う角度分布を持つ可能性がある（長尺は揃いやすい、短尺は乱れやすい）。Step 2 ではまず mean-field 近似から始める。

### 角度分布から G 比への変換
`orientation.py` に既に実装済みの Kawai 式を使う。P(θ) を渡せば I_∥/I_⊥ が出る。これは Phase 1 の遺産で、依然として正しい。

## テスト方針

### Step 1（単分散 2D Onsager）で必須のテスト
- 自由エネルギー F(b, S) が S=0 で常に極値（等方解は常に解）
- b < b_c で等方解が安定（S=0 が大域最小）
- b > b_c で nematic 解が安定（S>0 が大域最小）
- b → ∞ で S → 1
- ρ_c × L × d が L によらずほぼ一定（無次元化の確認）
- 与えられた ρ, L, d で S を出す関数の数値安定性

### Step 2（多分散）以降で追加するテスト
- 単分散リミット（σ=0）で Step 1 と一致
- 共存領域の存在確認（bidisperse で）
- 長尺優先 nematic 化の確認

### Step 1 の出力
- ρ × L × d vs S のグラフ（複数 d で重ね描き）
- 自由エネルギー曲面 F(S; b) の可視化
- ρ_c の数値出力（L=200, 300, 500 nm の各値で）

### Phase 1 で得た重要な知見（Phase 2 でも有効）

3 サンプルの長さ分布を比較すると、**ホモジ時間と長さ・配向度の関係は単調でない**：

| サンプル | median (nm) | L≥334 nm 本数比 | 実験配向度 |
|---|---|---|---|
| 8 min | 177 | 19% | **1.78** |
| 15 min | **236** | **33%** | 1.59 |
| 30 min sup | 189 | 27% | 1.20 |

8 min が一番短いのに配向度が一番高い。長さだけでは決まらない。Phase 2 ではこれを **「面密度・単離度・分布幅の相互作用」** として説明する。

## 既知の未解決問題（Phase 2 で取り組む）

1. **配向度 3.4 が出た理由**: 短尺リッチ（L小） + 高単離度（d 小）+ 高面密度の組み合わせで Onsager 閾値を超え、鋭い nematic 形成が起きた可能性
2. **最適 drop 間隔の存在理由**: 蓄積動力学の最適点（速すぎる: 相転移する前に通過、遅すぎる: 蒸発・解離など別効果）
3. **真空 2.5 で配向度 1.1 になった理由**: 速すぎて 2D 蓄積の前にポアを通過 or 流速で乱される
4. **8 min が最強で 15 min/30 min が劣る理由**: 単離度・分布幅・面密度の組み合わせ
