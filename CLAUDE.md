# CNT-CVF Simulation - Claude Code 開発ガイド

## プロジェクト概要

真空ろ過法（CVF: Centrifugal Vacuum Filtration）によるCNT配向プロセスを流体力学的にモデル化するシミュレーション。実験条件（drop間隔・膜構造・CNT長さ分布）と配向度（ラマンGピーク比）の関係を再現することが目的。

最終的に出したい答え：
1. **長さ分布の最適形**（中央値・幅・カットオフ）
2. **流速（drop 間隔）の最適値とその物理的理由**
3. **真空度との関係**
4. 未解明事項（短尺スタートで 3.4、真空 2.5 で 1.1）の説明

つまり「どうすればもっと配向度を上げられるか」を物理ベースで言える状態にする。

## Step 1〜2 で得た知見（重要）

### Step 1（Kawai 式）
- W=334 nm, L=230 nm（実験条件）では **L<W のため配向度 = 1.0**
- 実験値 1.78 とのギャップは **想定通り**。Kawai 式単独では説明できない
- L>W の長尺成分（長さ分布の上側 ~25%）と流体配向の組み合わせで説明する必要

### Step 2（Langevin、単純せん断）
- 実験条件での Péclet 数 **Pe ≈ 0.04**（境界層せん断 γ̇ = 66 s⁻¹、L=230 nm）
- Pe ≪ 1 → 熱揺動支配 → シミュ上の配向度はほぼ 1.0
- **「膜上の弱い境界層せん断」だけでは観測されている配向は絶対に作れない**
- 配向の主要因は他にある。候補：
  1. **ポア入口の局所的な強せん断**（バルクの 45 倍速い、γ̇ ~ 3000 s⁻¹、Pe ~ 2）
  2. **長尺 CNT の選択的寄与**（D_r ∝ L⁻³、L=1000 nm なら同条件で Pe ~ 2.6）
  3. **溝の幾何学的拘束**（L>W の成分のみ）
- → Step 3 では γ̇ を「単一の代表値」で扱うのをやめ、**γ̇(r, z) として書く**

### CVF で予想されるせん断率の階層

| スケール | 場所 | γ̇ [s⁻¹] | Pe (L=230 nm) |
|---|---|---|---|
| バルク | 膜から離れた所 | ~0 | ~0 |
| 境界層 | 膜表面付近 | ~66 | 0.04 |
| ポア間ピッチ | 隣接ポア間 (~600 nm) | ~500 | 0.3 |
| ポア入口収束 | ポア直上 (~100 nm) | ~3000 | 2 |

## Step 3 のスコープ（再設計版）

### 何を実装するか

**Step 3-A: 流れ場モデル**
- ポア入口の収束流を解析的に書く（点シンク or Sampson 流近似）
- 膜上の位置（ポアからの距離 r、高さ z）の関数として **γ̇(r, z)** を返す
- ポア配置はランダム or 規則格子（密度 2.8 × 10¹² 個/m²）
- 関数として実装：`local_shear_rate(r, z, U_macro, pore_geometry) → γ̇`

**Step 3-B: モンテカルロ**
- N 本の CNT について：
  1. 長さを **empirical CDF からサンプル**（`data/length_distributions/` の CSV）
  2. 位置（膜上のどこに居るか）をサンプル
  3. その位置・高さでの γ̇ を Step 3-A から取得
  4. Step 2 の Langevin で定常配向 θ_∞ を計算
- N 本の角度分布から Kawai 式で G ピーク比を出す
- **長さ重みづけ**で集計（ラマンは体積で重みづけられる）

**Step 3-C: sanity check**
- 8 min サンプル分布 → 1.78 が再現できるか
- 15 min サンプル分布 → 1.59 が再現できるか
- 完全には合わないはず（溝拘束がまだ無いので）。**どれくらい外すか**が次の手がかり

### Step 3 で入れない物理（Step 4-5 に回す）
- 溝壁の幾何学的拘束（位置に応じた θ_max カットオフ）
- 堆積層の効果
- CNT 間相互作用
- 膜の不均一性（中心部 vs 外周部）

### 主要な物理量と式

#### 並進拡散係数
```
D_trans = kT × ln(L/d) / (2πηL)
        ≈ 1.6 × 10⁻¹¹ m²/s
```

#### 回転拡散係数（Broersma）
```
ζ_r = π η L³ / (3 × (ln(L/d) - 0.66))
D_rot = kT / ζ_r
τ_rot = 1 / (2 D_rot) ≈ 0.31 ms (L=230 nm)
```

#### Péclet 数
```
Pe = γ̇ / D_rot
```
Step 2 で確認済み: 境界層 γ̇=66 s⁻¹、L=230 nm で **Pe = 0.04**。

#### Stokes抗力（ロッド、流れに垂直）
```
F = 4πηLU / [ln(L/d) + 0.5]
F ≈ 数 fN @ U = 数 µm/s
```

#### Kawai式（配向度計算）

溝内のCNT角度分布を一様分布と仮定（θ ∈ [-θ_max, θ_max]）：

- L > W: θ_max = arcsin(W / L)
- L ≤ W: 自由回転（配向なし）→ Gピーク比 = 1.0

Gピーク比（解析形）:
```
I_∥/I_⊥ = (11 θ_max + (9/2) sin(2θ_max)) / (11 θ_max - (9/2) sin(2θ_max))
```

または直接積分:
```
I_∥/I_⊥ = ∫[-θmax, θmax] (10cos²θ + sin²θ) dθ
         / ∫[-θmax, θmax] (cos²θ + 10sin²θ) dθ
```

Step 3 では一様分布の代わりに **MC で得た θ の経験分布** を Kawai 式の積分核に
入れて G ピーク比を計算する。

## コーディング規約

### 単位系
- **すべて SI 単位（m, s, kg, K, Pa, N）で内部計算する**
- nm, µm, fN, ms などの単位は表示時のみ変換
- 変数名に単位サフィックスを付けない（`length` は m 単位、と決め打ち）。ただし docstring には必ず単位を明記

### 型ヒント
- 全関数に型ヒントを付ける
- numpy配列は `np.ndarray` で型ヒント、shape は docstring で明記

### Docstring
- 各関数・クラスに docstring 必須
- パラメータ・戻り値の **物理的意味と単位** を明記
- 数式が出てくる場合は LaTeX 記法で docstring に含める

### 関数設計
- 1関数1物理プロセス
- 純粋関数を優先（副作用を避ける）
- 物理パラメータはハードコードせず、関数引数または config から読む

## やってはいけないこと

1. **物理パラメータを勝手に「より現実的な値」に書き換えない**
   - 例: η を「DOCだから水より粘度が高いはず」と勝手に変更しない
   - 値を変更したい場合は必ずユーザーに確認
2. **モデルを勝手に「改善」しない**
   - 各 Step で要求されているものだけ実装する
3. **ユーザーが書いた物理コメントや数式を消さない**
4. **単位を混在させない**
   - すべて SI で統一。nm, µm を内部計算で使わない
5. **エラーハンドリングで物理的に不正な値を黙って通さない**
   - 例: L<0, W<0, U<0, T<0 などは raise する
6. **テストを省略しない**
   - 各モジュールに物理的下限値のユニットテストを書く
7. **「既存実装と矛盾するからこっちを変えました」を勝手にやらない**
   - 矛盾を見つけたら報告して、どちらが正しいかユーザーに確認する
8. **長さ分布データを勝手に加工しない**
   - 100 nm カットオフはデータ取得時に既にかかっている。再カットしない
   - 102.4 nm の張り付き（人工ピーク）もそのまま使う
   - フィッティング（log-normal）を経由せず、empirical CDF から直接サンプル

## テスト方針

### 物理的に既知の極限でユニットテスト

#### Step 1（Kawai式）✓
- L < W → 配向度 = 1.0（厳密）
- L = W → 配向度 = 1.0（境界）
- L >> W → θ_max → 0、配向度 → 10（係数の上限）

#### Step 2（Langevin）✓
- D_rot が L³ に逆比例
- Pe = 0 で <cos 2θ> → 0（等方化）
- Pe → ∞ で θ が安定点に収束

#### Step 3（MC + 流れ場）
- Step 3-A: γ̇(r, z) の積分で全流量が保存（連続の式）
- Step 3-B: 全 CNT が同じ短い長さ（L < W）の極限で配向度 → 1.0
- Step 3-B: γ̇ → 0 の極限で配向度 → 1.0
- Step 3-B: 長さ分布が単一の長尺値の極限で、Kawai 式単独の予測と一致

### 直感に反する事実

3 サンプルの長さ分布を比較すると、**ホモジ時間と長さの関係は単調でない**：

| サンプル | median (nm) | L≥334 nm 本数比 |
|---|---|---|
| 8 min | 177 | 19% |
| 15 min | **236** | **33%** |
| 30 min sup | 189 | 27% |

そして配向度は逆に 8 min（1.78）> 15 min（1.59）。
**長尺成分が多いほど配向度が高いわけではない** → 長さ以外の因子（流速・濃度等）も
効いている。Step 3 〜 4 で定量的に分離する。

## 既知の未解決問題（シミュレーションで解明したい）

1. **配向度3.4が出た理由**: 短尺スタート材料での突出した値。長さ分布の均一性 × 流速の共鳴か？
2. **最適drop間隔の存在理由**: 速すぎても遅すぎても配向度が崩壊する機構
3. **真空2.5で配向度1.1になった理由**: 捕捉はされているのに配向しない
4. **8 min vs 15 min**: 15 min の方が長尺リッチなのに配向度が低い理由
